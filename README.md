# otterai-api

Unofficial Python API for [otter.ai](http://otter.ai)

## Installation

`pip install .`

or in a virtual environment
 ```bash 
 python3 -m venv env
 source env/bin/activate
 pip install .
 ```
 
 ## Setup
 ```python
 from otterai import OtterAI
 otter = OtterAI()
 otter.login('USERNAME', 'PASSWORD')
 ```
